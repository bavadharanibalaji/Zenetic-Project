🏥 Zenetic – Healthcare Web Application

📌 Project Description

Zenetic is a responsive healthcare web application designed to provide users with an easy and efficient way to explore medical services and book appointments with doctors. The platform focuses on improving user experience through a clean interface, real-time search functionality, and interactive UI components.

---

🚀 Features

- 🔍 Doctor Search Functionality – Users can search for doctors and specialties instantly
- 📅 Appointment Booking – Simple booking interface with confirmation alerts
- 📊 Health Dashboard – Displays health metrics like heart rate, blood pressure, etc.
- 📱 Responsive Design – Works on mobile, tablet, and desktop
- 🎯 Interactive UI – Includes animations, hover effects, and dynamic updates
- 💡 Health Tips Section – Displays random health tips for users

---

🛠️ Technologies Used

- HTML5
- CSS3 (Responsive Design & Animations)
- JavaScript (DOM Manipulation & Interactivity)
